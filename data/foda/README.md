# foda — FODA institucional 2025

Esta fuente alimenta la slide **FODA Institucional** (4 cuadrantes).

## Cómo editar

1. Abre `foda.xlsx`.
2. Filtra por `cuadrante` para ver solo el cuadrante que vas a editar.
3. Edita `item` o añade filas nuevas con el `orden` correspondiente.
4. Guarda.

## Estructura

- 4 cuadrantes × 7 ítems = 28 filas totales (sin contar encabezado).
- Cada ítem es una frase declarativa corta (1 línea).
