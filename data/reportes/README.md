# reportes — Reportes anuales e histórico

Esta fuente alimenta la slide **Reporte Anual Institucional** con histórico de cumplimiento y KPIs vigentes.

## Cómo editar

1. Abre `reportes.xlsx`.
2. Para añadir un nuevo año al historial: nueva fila con `anio`, `planes`, `cumplimiento`.
3. Para actualizar KPIs del año actual: edita las celdas en la sección `kpis`.
4. Guarda.

## Estructura

- **Sección 1** (filas 1-4): historial por año.
- **Marcador** `-- hoja: kpis --`: separador visual.
- **Sección 2** (siguientes filas): KPIs institucionales.
