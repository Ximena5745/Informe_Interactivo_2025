# objetivos — Objetivos estratégicos (CMI)

Esta fuente alimenta la slide **Objetivos Estratégicos** del Cuadro de Mando Integral.

## Cómo editar

1. Abre `objetivos.xlsx`.
2. Edita el `real` para actualizar el avance del año.
3. Para añadir un objetivo: nuevo ID correlativo `OE-13`, etc.
4. `linea` debe existir en `avance.xlsx`.
5. Guarda.

## Perspectivas CMI

- **Financiera**: sostenibilidad económica.
- **Clientes/Mercado**: estudiantes, sector productivo, sociedad.
- **Procesos internos**: calidad, eficiencia, transformación.
- **Aprendizaje**: conocimiento organizacional, capacidades.
